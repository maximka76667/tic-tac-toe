# Tic Tac Toe  

## Description
Tic Tac Toe game. Two game modes are available: singleplayer against a bot and multiplayer for 2 players.  
Here I tried TypeScript for the first time to create a simple game.

## Tech Stack
`React.js` `TypeScript`
